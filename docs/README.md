# 羽毛球报名小程序 - Java后端迁移计划

## 项目概述

本项目是将现有微信小程序云开发后端迁移到Java Spring Boot后端的完整方案。

### 当前架构
```
微信小程序 → 微信云函数(28个) → 云数据库
```

### 目标架构
```
微信小程序 → Java Spring Boot API → MySQL数据库
```

## 快速开始

### 环境要求
- JDK 17+
- MySQL 8.0+
- Maven 3.8+
- IDE: IntelliJ IDEA

### 技术栈
| 组件 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 3.2.x | 应用框架 |
| MyBatis-Plus | 3.5.x | ORM框架 |
| MySQL | 8.0 | 数据库 |
| WxJava | 4.5.x | 微信SDK |
| JWT | - | 用户认证 |
| Lombok | - | 代码简化 |

## 迁移范围

### 需要迁移的模块（28个云函数）

| 模块 | 云函数数量 | 优先级 |
|------|-----------|--------|
| 用户模块 | 4 | P0 |
| 活动模块 | 5 | P0 |
| 报名模块 | 3 | P0 |
| 比赛模块 | 4 | P0 |
| 分组模块 | 1 | P1 |
| 挑战赛模块 | 4 | P1 |
| 排名模块 | 1 | P1 |
| AI模块 | 2 | P2 |
| 其他模块 | 4 | P2 |

### 不需要迁移的部分
- 微信登录（使用微信SDK直接实现）
- 文件上传（改用OSS/MinIO）
