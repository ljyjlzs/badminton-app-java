```bash
mvn spring-boot:run
```

启动成功后会看到：

应返回规则数据的 JSON。

2. 导入项目：选择 `badminton-app-java/miniprogram` 目录
3. AppID 填写：`wxb0369110c3c788ef`
4. 在开发者工具中点击「编译」即可预览

### 4. 测试流程

1. **登录测试**
   - 打开小程序，点击授权登录

## 三、部署上线

### 1. 部署后端

#### 方式一：直接运行 JAR

```bash
# 打包
cd D:/project/opencode/badminton-app-java
mvn clean package -DskipTests

# 运行（生产环境）
java -jar target/badminton-app-0.0.1-SNAPSHOT.jar --spring.profiles.active=prod
```

#### 方式二：使用 systemd（Linux 服务器）

创建 `/etc/systemd/system/badminton.service`：

```ini
[Unit]
Description=Badminton App
After=syslog.target network.target

[Service]
User=www
ExecStart=/usr/bin/java -jar /opt/badminton/badminton-app.jar --spring.profiles.active=prod
SuccessExitStatus=143
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

启动服务：
```bash
sudo systemctl daemon-reload
sudo systemctl start badminton
sudo systemctl enable badminton
```

### 4. 配置 Nginx 反向代理

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # HTTP 重定向到 HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl;
    server_name your-domain.com;

    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;

    location /api/ {
        proxy_pass http://localhost:8049;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

## 四、常见问题

### 1. 数据库连接失败
检查 MySQL 服务是否启动，用户名密码是否正确。

### 2. 微信登录失败
检查 AppID 和 AppSecret 是否配置正确。

### 3. 401 未登录错误
检查 JWT 密钥是否一致，token 是否过期。

### 4. 跨域问题
后端已配置 CORS，如果仍有问题检查 Nginx 配置。
