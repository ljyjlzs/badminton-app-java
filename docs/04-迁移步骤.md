**分组算法**：
```java
/**
 * 均衡分组算法
 * 1. 按等级降序排列
 * 2. 最高等级+最低等级配对
 * 3. 重复步骤2直到所有人分配完毕
 * 4. 将队伍按总等级降序排列
 * 5. 头尾配对、中段配对，形成比赛
 */
public List<Team> groupByBalance(List<Registration> registrations) {
    // 排序
    registrations.sort((a, b) -> b.getLevel() - a.getLevel());

    // 配对
    List<Team> teams = new ArrayList<>();
    int left = 0, right = registrations.size() - 1;
    while (left < right) {
        Team team = new Team();
        team.setMembers(Arrays.asList(
            registrations.get(left).getUserId(),
            registrations.get(right).getUserId()
        ));
        teams.add(team);
        left++;
        right--;
    }

    return teams;
}
```

**积分计算**：
```java
/**
 * 计算积分
 */
public int calculateScore(int myScore, int opponentScore, String round) {
    if ("group".equals(round)) {
        // 小组赛：胜者+分差，败者-分差
        return myScore - opponentScore;
    } else if ("challenge".equals(round)) {
        // 挑战赛：胜者+10，败者-10
        return myScore > opponentScore ? 10 : -10;
    } else if ("final".equals(round)) {
        // 决赛：胜者+15，败者-15
        return myScore > opponentScore ? 15 : -15;
    }
    return 0;
}
```

## 第三阶段：前端改造（3-5天）

### 3.1 封装HTTP请求工具

创建 `utils/request.js`：
```javascript
const BASE_URL = 'https://your-domain.com/api';

const request = (options) => {
  return new Promise((resolve, reject) => {
    const token = wx.getStorageSync('token');
    wx.request({
      url: BASE_URL + options.url,
      method: options.method || 'GET',
      data: options.data,
      header: {
        'Authorization': token ? `Bearer ${token}` : '',
        'Content-Type': 'application/json'
      },
      success: (res) => {
        if (res.data.code === 200) {
          resolve(res.data);
        } else if (res.data.code === 401) {
          // token过期，重新登录
          wx.navigateTo({ url: '/pages/index/index' });
        } else {
          reject(res.data);
        }
      },
      fail: reject
    });
  });
};

module.exports = { request };
```

### 3.2 替换云函数调用

**示例**：
```javascript
// 之前
wx.cloud.callFunction({
  name: 'get-activities',
  data: { type: 'available' },
  success: res => { ... }
});

// 之后
const { request } = require('../../utils/request');

   mvn clean package

   # 2. 上传jar包到服务器

   # 3. 启动
   java -jar badminton-app.jar --spring.profiles.active=prod

   # 4. 配置Nginx反向代理
   ```

4. **Nginx配置**
   ```nginx
   server {
       listen 443 ssl;
       server_name your-domain.com;

       ssl_certificate /path/to/cert.pem;
       ssl_certificate_key /path/to/key.pem;

       location /api {
           proxy_pass http://localhost:8080;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
       }
   }
   ```
